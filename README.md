## Overview

Team Balancer is a Flask web application that distributes players or participants into evenly matched teams based on their skill ratings. Useful for sports tournaments, gaming sessions, or any scenario requiring fair team formation.

## Features

- Skill-based team balancing algorithm
- Configurable for sports, gaming, or project groups
- Simple and intuitive web interface
- SQL-backed player and team data

## Getting Started

**Prerequisites:**
- Python 3.x
- pip

**Installation:**
```bash
git clone https://github.com/Yoavsb25/TeamBalncer.git
cd TeamBalncer
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000
